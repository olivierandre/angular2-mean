## Installation


#### Dependencies

- Bower
```
$ bower install
```

- NPM
```
$ npm install
```

Copy angular2 & systemjs folder to /public/lib

#### Start
- Launch server (dev mode)
```
$ gulp server
```

#### Usage
> go to localhost:3000 or localhost:1337
